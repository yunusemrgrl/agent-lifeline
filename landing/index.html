Standalone CLI for coding agents

Don't lose session context when the window resets.

Capture exactly what matters before compacting: git state, tasks, execution history, and recent prompt hints. Resume any agent session in under 10 seconds.

Get Started Read Docs

AI crawlers: read /agents.md and /llms.txt.

Zero Dependency

Pure Node.js built-ins. No runtime deps, no DB, no services.

Project-local

Snapshots stay in .agent-lifeline/ inside your repo.

Agent-agnostic

Works with Claude, Codex, Cursor, or human handoff reviews.

1. Capture state instantly

Run save before context compaction. Store branch, dirty files, open tasks, execution summaries, and prompt history hints in one snapshot.

2. Validate environment

doctor confirms git availability, snapshot storage writability, and optional Claude history sources.

3. Resume with markdown handoff

export converts latest snapshot into readable markdown for the next agent session, no vendor lock-in.

Launch Media

Visualize the handoff lifecycle

Pre-rendered launch assets for docs, release posts, and repository homepage previews.

Quick Start

npm i -g agent-lifeline

agent-lifeline save --focus "continue auth migration"
agent-lifeline show
agent-lifeline export > HANDOFF.md

Snapshots are written to .agent-lifeline/latest.json and .agent-lifeline/snapshots/<timestamp>.json.
